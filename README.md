## 📌 Descrição

Este projeto foi desenvolvido para estimar o gasto de energia elétrica de um ar-condicionado de forma simples e prática.

O sistema recebe como entrada:
- Capacidade em BTUs  
- Tempo de uso diário  
- Tipo do equipamento (convencional ou inverter)  

Com base nesses dados, calcula:
- Consumo mensal (kWh)  
- Custo mensal (R$)  
- Nível de consumo (baixo, médio ou alto)  

Além disso, apresenta um comparativo entre modelos convencional e inverter, destacando a economia gerada.

## 🧮 Fórmula

Consumo = (potência × horas × 30) / 1000  

Conversão:  
1 BTU ≈ 0,293 W

## 💡 Exemplo

Entrada:
- 12000 BTUs  
- 8 horas/dia  
- inverter  

Saída aproximada:
- Consumo: ~420 kWh/mês  
- Custo: ~R$ 378  
- Nível de consumo: médio  
- Economia: ~R$ 380/mês

## 📊 Observação
